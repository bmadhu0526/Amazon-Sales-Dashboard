## Overview
This is a self project I built to strengthen my data analytics skills as a fresher.  
The project uses a raw Amazon sales dataset and converts it into a complete, interactive Power BI dashboard to analyze sales performance, profit trends, shipping behavior, customer segments, and product categories.

## What I Did
- Cleaned and transformed the dataset using Power Query
- Created KPI cards for Sales, Profit, Orders, and Product Count
- Built visuals for Ship Mode, Segment, Category, and Return Status
- Added Year-wise comparison between 2019 and 2020
- Designed US Geo Map for region-wise sales analysis
- Created Quarterly Sales vs Profit visualization
- Identified top profitable sub-categories

## Key Insights
- Total Sales: 1.57M  
- Total Profit: 0.18M  
- Orders: 5901  
- Top Shipping Mode: Standard Class  
- Most Profitable Sub-Categories: Phones and Binders  
- Best Selling Category: Office Supplies
